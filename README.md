# Rosalind Problems

Solutions to [Rosalind](https://rosalind.info/) bioinformatics problems in Rust.

## Usage

```sh
cargo run --bin <problem_id>
```

## Problems

| ID | Problem | Solution |
|----|---------|----------|
| dna | [Counting DNA Nucleotides](https://rosalind.info/problems/dna/) | [dna.rs](src/bin/dna.rs) |
| rna | [Transcribing DNA into RNA](https://rosalind.info/problems/rna/) | [rna.rs](src/bin/rna.rs) |
| recv | [Complementing a Strand of DNA](https://rosalind.info/problems/revc/) | [revc.rs](src/bin/revc.rs) |
